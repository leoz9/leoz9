## 🔥 Featured Work

### 🤖 Multi-modal Agent System

> Production-oriented LLM Agent with workflow orchestration & tool-use

#### 🧠 Capabilities
- Multi-modal input: **text / speech / image**
- Long-horizon reasoning via **LangGraph**
- Tool calling (APIs / RAG / external systems)
- Stateful planning & execution

#### ⚡ System Design
- FastAPI service layer (async)
- Modular agent pipeline (planner → executor → tools)
- Extensible workflow graph

#### 🚀 Inference Optimization
- vLLM high-throughput serving
- INT8 / INT4 quantization
- Memory & latency optimization
