## 🔥 Featured Project

### 🤖 多模态 Agent 智能化客服
- 基于 **LLM + LangGraph + FastAPI** 打造的智能客服系统
- 支持 **文本、语音、图片** 等多模态输入
- 提供灵活的 **对话流程编排**，可扩展知识库与外部 API
- 使用 **PyTorch** 进行模型微调与推理加速
- 面向企业客服场景，支持 FAQ、任务型对话与复杂多轮交互
